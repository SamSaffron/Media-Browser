Media Browser Features

 

This page outlines some of the features of Media Browser. Due to the beta status of Media Browser version 2, not all features presented may be functionally complete.

Automatic Metadata Retrieval

Media management comes alive with Media Browser, with the ability to automatically search online sources for fetch posters, artwork, fanart, synopsis, cast and much more for both movies and tv episodes.Media Browser leverages online data providers theMovieDb.org and tvDb.com for movie and television metadata, artwork respectively.

In addition Media Browser integrates with several established metadata catchers allowing for finer control of metadata with your media. Some metadata applications include:

  • Media Scout
  • Metabrowser
  • Salami's Movie Organizer
